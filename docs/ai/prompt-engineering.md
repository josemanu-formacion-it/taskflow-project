# Prompt Engineering

## ¿Qué voy a documentar aquí?

Este documento recoge las técnicas de prompt engineering aplicadas durante la Fase 2
para obtener mejores resultados de los asistentes de IA.

Incluirá:
- Principios básicos del prompt engineering
- Prompts utilizados durante el desarrollo (buenos y malos)
- Análisis de por qué funcionan o fallan ciertos prompts
- Plantillas reutilizables para tareas comunes de desarrollo
- Evolución y mejora de prompts a lo largo del ejercicio
